# guide/images

콘텐츠/안내용 이미지 자산 디렉터리.

## 디테일 향상 팁 팝업 (create.html)

`create.html`의 "더 완성도 높은 도안으로 만들려면?" 팝업이 아래 비교 이미지를
참조합니다. 파일이 없으면 회색 placeholder 박스 + 캡션이 표시되고, 아래 경로에
실제 파일을 올리면 자동으로 표시됩니다. (코드 수정 불필요)

| 예약 파일명 | 내용 |
|---|---|
| `detail-tip-crop.png` | 전신 크롭 도안 vs 얼굴 크롭 도안 비교 |
| `detail-tip-size.png` | 작은 캔버스 vs 큰 캔버스 얼굴 부분 확대 비교 |

- 권장 비율: 16:9 (팝업의 figure가 `aspect-ratio: 16/9`로 자리를 잡음)
- 운영자가 사이트에서 직접 생성·캡처해 넣는 자산입니다.
