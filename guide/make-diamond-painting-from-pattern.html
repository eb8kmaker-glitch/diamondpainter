From Pattern to Finished Piece — Making Diamond Painting at Home

In this guide

  • Before You Start
  • What You'll Need
  • Understanding the Chart
  • Step by Step
  • Good to Know

Before You Start

Diamond Painter generates a printed symbol chart, not a pre-glued canvas with symbols printed on it like store-bought kits. This creates the key difference in how you work.

  • Store-bought kit: You place each diamond directly onto its symbol on the canvas.
  • Printed chart + blank canvas: You read the coordinates on the paper chart and transfer each diamond, cell by cell, onto a separate blank sticky canvas — the same way you read a chart and stitch onto fabric in cross-stitch.

A home printer can't replicate a store-bought kit exactly, but if you prepare a blank sticky canvas separately, you can create the same result yourself.

What You'll Need

  • Blank sticky canvas — a blank canvas with a printed grid and an adhesive surface. Search for "blank diamond painting canvas." Choose the grid (square or round) to match the diamond type you'll use.
  • Diamonds (drills) — prepare them by color, following the DMC numbers in the chart's color legend.
  • Pen, tray, and wax — the basic tools for picking up and placing diamonds.
  • Your printed chart — the paper chart this guide is based on.

Understanding the Chart

The printout is usually split across several pages. A 143×143 chart, for example, is too large for one sheet, so it's divided into sections like A1, A2, B1, B2. The top of each page shows the coordinate range for that section (e.g., col 1–76, row 1–105).

  • Grid coordinates — every cell's position is fixed by its column (col) and row numbers. Find the same coordinate on the canvas and place the diamond there.
  • Symbol — the shape in each cell tells you which color diamond it is.
  • Color legend — a table pairing symbol ↔ DMC number ↔ color name. It also shows the number of colors, the count and percentage of diamonds per color, the total diamond count, and the finished real size (e.g., 40×40cm). Use it to see how many diamonds of each color to prepare.

Step by Step

1

Sort diamonds using the color legend. Separate them by DMC number into trays and match them to the symbols on the chart.

2

Choose a starting section. Usually begin from a corner page (e.g., A1) and align its coordinate origin with the canvas.

3

Place cell by cell along the coordinates. Check the symbol in a cell on the chart → pick up the matching DMC color → place it on the same coordinate on the canvas. Filling much of one color before moving to the next reduces tray swaps and works more efficiently.

4

Work section by section. Following an order like A1 → A2 → B1 → B2, keeping the coordinates continuous across page boundaries, helps avoid misalignment.

5

Finish. Once everything is filled, press gently on a flat surface to set the diamonds.

Good to Know

  • Prepare extra diamonds. The per-color counts in the color legend are a baseline. Since some get dropped or come defective, it's safer to prepare a little extra of each color.
  • Dye lot variation. Even the same DMC number can have slight color differences depending on the production batch. When possible, prepare a single color from the same lot.
  • Check your coordinates often. Working across split pages makes it easy to drift by one cell at the boundaries. Periodically check your position against the thick gridlines (usually every 10 cells).
