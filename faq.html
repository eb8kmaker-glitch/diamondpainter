Images and Files

Diamond Painter accepts JPG/JPEG, PNG, and WebP image files. Most photos from smartphones and cameras are in one of these formats. Very large files (above 20MB) may take a moment to load in the browser, but will work correctly.
High-contrast images with a clear main subject produce the best results. Pet portraits, single-person portraits, simple landscapes, and graphic designs with clear color areas convert very well. Very busy images, low-light photos, or images with lots of fine details (such as a crowd or a complex cityscape) can be more challenging — try reducing the color count for these images.
Use the Save button in the tool to download a project file (.json format) to your device. To continue your project, open the tool again and use the Load button to reload your saved file. Your pattern, color choices, and any manual edits you have made will be restored.
The export function downloads a high-resolution PNG image of your current canvas view — including the grid, symbols, and color coding. You can print this image as a paper pattern to reference while working, or provide it to a custom canvas supplier along with your color list.

Pattern Generation

The tool uses a perceptual color clustering algorithm to reduce your image to the number of colors you select. Each cluster color is then matched to the nearest DMC color in our database of 457 authentic DMC drill shades using perceptual color distance (which accounts for how human vision perceives color differences). The result is that each color in your pattern maps to a real, orderable DMC code.
For most images, 25–40 colors is a good starting point. Simple images (silhouettes, graphic designs, simple landscapes) work well with 15–25 colors. Complex images like detailed portraits can benefit from 35–50 colors. Going above 50 colors rarely improves the visual result noticeably and makes the project more expensive and harder to organize.
Preset sizes include A4, A3, 30×40cm, 40×50cm, 50×70cm, and 50×80cm. You can also set completely custom dimensions using the custom size option. The tool displays the resulting grid size and approximate bead count for each combination of canvas size and drill type in real time.
Round drills (2.8mm) are easier to place and more forgiving of small errors, but leave small visible gaps between drills in the finished piece. Square drills (2.5mm) fit together without gaps, creating a seamless mosaic-like finish that looks more complete. Square drills are more challenging to place precisely but are preferred by many experienced crafters for display pieces.

Diamond Painting Questions

Search for "diamond painting drills DMC [code number]" on Amazon, Etsy, or dedicated diamond painting suppliers. Most suppliers sell individual color bags in quantities of 200, 500, or 1000 drills. Because DMC codes are standardized, you can mix suppliers freely — the same code from different suppliers will be the same color (with minor variation between manufacturers).
Yes. Export your pattern image and save your DMC color list, then approach a custom diamond painting canvas supplier. Many suppliers on Etsy and specialized crafting sites offer a service where you upload your image or pattern and they print an adhesive canvas with the symbols pre-printed. Provide your DMC color list so the supplier knows which drill bags to include in the kit.
Export your pattern as a PNG file, then print it at your preferred scale. For large canvases, print across multiple A4 or letter sheets and tape them together. Print shops can also produce a single large-format print, which is cleaner to work from. Having a printed reference is especially useful if you are working from a paper pattern rather than a pre-printed canvas.
